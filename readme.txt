== Installation ==

1. Upload the entire `ws-analytics` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Tools' -> 'Analytics' in WordPress.
4. Now checked is active checkbox.
5. Add your Google Analytics tracking code here.
